# Manhunt Resource Pack:
This repository contains the official resource pack used by the *Manhunt plugin*.
The resource pack is required to load and display custom models, items, and GUIs that are used by the plugin during gameplay.
Without this resource pack, custom items and GUIs provided by the plugin may not render correctly.

## Purpose:
### The Manhunt Resource Pack is used to:
1. Import custom item models
2. Render custom ability items
3. Display custom GUI textures
4. Ensure consistent visuals across all players using the Manhunt plugin

## Plugin Dependency:
This resource pack is specifically designed for and used by the Manhunt Minecraft plugin.

1. Plugin: Manhunt
2. Compatibility: Spigot / Paper
3. Usage: Automatically servers running the Manhunt plugin

## Installation:
This resource pack is automatically applied to the clients if the Manhunt Plugin is installed in the server.
